# 00 — Brand Input

> **Nơi đặt tài liệu brand của Trung Tâm Tiếng Trung Khánh Linh.**

## Cần user cung cấp:

1. **Logo** (file `.png` / `.svg`) — bản chính + bản đảo màu nếu có
2. **Bảng màu** — màu chính, màu phụ, màu accent (mã HEX hoặc ảnh palette)
3. **Font chữ** — tên font dùng cho tiêu đề và body (cả tiếng Việt + tiếng Trung)
4. **Ảnh tham khảo / moodboard** (tùy chọn) — design nào đang thích, muốn theo phong cách gì
5. **Tone & voice** (tùy chọn) — giọng văn muốn dùng: nghiêm túc / trẻ trung / hàn lâm…
