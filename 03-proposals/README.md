# 03 — Proposals

> **Nơi chứa từng proposal cho khách hàng cụ thể.**

## Quy ước đặt tên folder:

```
03-proposals/
├── 2026-05-khanh-linh-q3/         ← năm-tháng-tên_khách-mục_đích
├── 2026-06-abc-launch/
└── ...
```

## Cấu trúc mỗi folder proposal:

```
2026-05-khanh-linh-q3/
├── brief.md                 ← copy từ 02-templates/brief-template.md
├── strategy.md              ← copy từ 02-templates/strategy-template.md
├── proposal-content.md      ← viết content text trước
├── proposal.html            ← build từ proposal-template.html
├── pricing.xlsx             ← báo giá + timeline
└── assets/                  ← ảnh, logo khách, mockup
```

## Workflow:

→ Mỗi proposal đi qua 5 bước với 5 checkpoint. Xem chi tiết trong `PLAN.md` mục **3. Workflow**.
